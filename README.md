#### Description
Simple example of how to integrate TornadoFx in a String Boot application.

#### How to build
###### Unix
```bash
./gradlew bootRepackage
```
###### Windows
```bash
> gradlew.bat bootRepackage
```


#### How to run
```bash
> java -jar build/libs/build/libs/spring-boot-tornadofx-0.0.1-SNAPSHOT.jar
```
